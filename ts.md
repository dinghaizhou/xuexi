### 调试环境
* 安装ts 

  ```
  npm i typescript -g
  ```

* 配置文件

  ```
  tsc --init
  ```

* 生成package.json

  ```
  npm init
  ```

* 工程化
	* 安装webpack，webpack-cli，webpack-dev-server
	
	  ```
	  npm i webpack webpack-cli webpack-dev-server html-webpack-plugin typescript ts-loader
	  ```
